# GymFlow ERP
GymFlow is an Admin-Only ERP designed to manage gym operations, HCM, and inventory.

## Tech Stack
Frontend: Vite + React (UI)\
Backend: MVC Framework (PHP/Custom)\
Database: MySQL (db.sql included)

## Project Structure
/backend: Database connections, Models, and Controllers.\
/frontend: React source code, hooks, and API services.\
db.sql: Core database schema.

## Features
HCM: Staff attendance, roles, and status management.\
Inventory: Equipment tracking, brands, and warranty management.
